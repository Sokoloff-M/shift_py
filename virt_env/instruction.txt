# Инструкция по запуску сервиса

## Установка зависимостей
1. Убедитесь, что у вас установлен Python версии 3.x.
2. Установите зависимости с помощью команды:

pip install -r requirements.txt



## Запуск сервиса
1. Перейдите в директорию проекта.
2. Запустите приложение с помощью команды:

uvicorn main:app --reload


для Bash
Откройте браузер и перейдите на http://localhost:8000/docs, чтобы открыть документацию API.

## Взаимодействие с проектом
- Документация по API доступна на http://localhost:8000/docs. Здесь вы можете ознакомиться с доступными маршрутами и их параметрами.
- Вы можете использовать любой HTTP-клиент (например, curl или Postman) для отправки запросов к API и взаимодействия с сервисом.
